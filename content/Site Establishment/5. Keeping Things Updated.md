# Updating node.js (actual node)
user@User ~ % **node -v** -- show your currently installed version of node.js
user@User ~ % **nvm ls-remote** -- lists available versions
user@User ~ % **nvm install v22.21.0** -- install specified version
user@User ~ % **nvm install --lts** -- install lates LTS version
user@User ~ % **nvm use v22.21.0** -- indicates the nvm version you want to use
user@User ~ % **nvm uninstall v22.16.0** -- uninstall old versions

*Note:* I typically install the lates LTS version
# Updating npm (package manager)
user@User ~ % **npm -v** -- show your currently installed version of npm
user@User ~ % **nvm install-latest-npm** -- installs latest *compatible* version of npm

*Note:* It will only install the latest version which is compatible with the current nvm you have installed
