# Known issues

## Callouts
If your callouts are not functionally as expected (i.e. only partially collapsing), try the following:

1. In the *quartz.config.ts* file, under the *plugins > transformers section*, Add **Plugin.HardLineBreaks()** :

```ts
plugins: {
	transformers: [
		Plugin.HardLineBreaks(), // Added to address issues with translating line breaks in Obsidian to HTML
```
*Note:* This should also fix issues on line breaks not translating properly outside of callouts as well.

2. revert commit in terminal within quartz folder of your vault:
	- user@User quartz % **git revert 4923aff**
