## Install Quartz:
[Quartz](https://quartz.jzhao.xyz/) explains the installation process succinctly and may be all you need. Follow those instructions and come back here if you want more detail and nuance. 
- If you do skip this section in favor of Quartz's guide, feel free to jump to the [[4. Deploy your site with GitHub]] section once you have Quartz setup👋.
- For everyone else, please continue reading 🙃.

### Install Quartz in your Vault:
1. Navigate to your Vault directory using your terminal:
	- This command to change directories varies depending on your Operating System
	- Select the applicable drop-down example from the choices below:

>[!info]- macOS and Linux **change directory (cd)** example:
>- The following is an example format!
>- You'll need to change the directory names to match the path to your vault folder
>```bash
>cd ~/Documents/Vaults/"name of your vault folder"
>```
>![[Screenshot 2025-04-23 at 2.13.24 PM.png]]
>>[!tip]+ Alternatively:
>>- right click on your Vault folder and select **New terminal at folder** from the drop down.
>>![[Screenshot 2025-04-23 at 2.47.10 PM.png]]

>[!tip]- Windows **change directory (cd)** example:
>- The following is an example format! 
>- You'll need to change the directory names to match the path to your vault folder
>```bash
>cd C:\Documents\Vaults\"name of your vault folder"
>```

2. Enter the following command in your terminal to download Quartz into your Vault:
```bash
git clone https://github.com/jackyzha0/quartz.git
```

3. Navigate to the nested quartz sub-directory now located in your Vault:
	- This command to change directories varies depending on your Operating System
	- Select the applicable drop-down example from the choices below:

>[!info]- macOS and Linux **change directory (cd)** example:
>```bash
>cd quartz
>```

>[!tip]- Windows **change directory (cd)** example:
>- The following is an example format! 
>- You'll need to change the directory names to match the path to your vault folder
>```bash
>cd C:\Documents\Vaults\"name of your vault folder"\quartz
>```

4. Install applicable **node.js** packages for Quartz using the following terminal command:
```bash
npm i
```

5. Finish initializing the Quartz installation with this next command:
```bash
npx quartz create
```
